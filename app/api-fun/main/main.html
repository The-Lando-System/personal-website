This page was also my exploration in using Angular UI Bootstrap to manage views within views.

The graph shown in the GitHub tab is a JS library called Chart.js. It shows my commit activity to GitHub for this website over the past 10 weeks. I am able to get the data by making a GET AJAX call using $http to the repos/The-Lando-System/personal-website/stats/commit_activity endpoint.

In the NOAA Weather tab, I make a call to NOAAs weather service to return, among other things, a forecast for the weather at the provided latitude and longitude. Due to cross-origin resource sharing (CORS), I needed to use Angular's $jsonp call and add callback=JSON_CALLBACK to wrap my request in a callback function.

Additionally, I downloaded a CSV file that contains a mapping of latitudes and longitudes to zip codes and cities. When the user chooses a zip code, a request is made to retrieve the CSV file to parse out the latitude and longitude and make the NOAA request with it.

Codewars is a site where you can tackle various coding challenges in a variety of languages. You make a profile, join a "clan", and gain "honor" by completing challenges.

In order to retrieve profile data associated with my user, I attempted to use Angular's $http as well as $jsonp. The first way was denied due to CORS and the second way didn't fully work because the Codewars API didn't give me a function to execute with the callback I wrapped in the request. I ended up using Yahoo's YQL API as a proxy for making the JSONP request. I only had to supply the proxy server with a "query" parameter in the URL that contains the Codewars endpoint I'm interested in and it returned the data I wanted.
