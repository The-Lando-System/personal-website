Summary

This page displays sound data collected from my Arduino board via a Piezo transducer. The Piezo will convert a mechanical vibration to an analog signal which then passes through an ADC to output discrete values based on the frequency of vibration. When I plug the Arduino board to my computer, I can read the data on the serial port. I wrote a Python script to collect this data, and post it to an endpoint on this website at a configurable rate. I store the sound data in a Mongo DB back end, and display it in a JS visualization tool called Dynagraph.

The point of this project was two-fold: create a website for the first time, and combine it with an Arduino project.
